# REST API User Information

A simple web application that fetches and displays user data from a public REST API using JavaScript.

---

## Project Description

This project demonstrates how to integrate a REST API into a frontend application.  
User data is fetched dynamically and displayed in a responsive card layout.

A search feature allows filtering users by name, email, city, or company in real time.

---

## Features

- Fetches data from a public REST API
- Displays user information dynamically
- Real-time search and filtering
- Responsive grid layout
- Status messages for loading and errors
- Clean and readable user interface

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- REST API (fetch API)

## How It Works

1. The page loads and sends a request to a public REST API
2. User data is fetched using the `fetch` method
3. Data is displayed as cards on the page
4. The search input filters visible users in real time
5. Status messages inform the user during loading or errors
